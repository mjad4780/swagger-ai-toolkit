# Swagger Tools

Fetch OpenAPI/Swagger JSON from a URL and convert it to AI-ready Markdown.

## What it does
1. **Fetch**: Downloads `swagger.json` from any Swagger UI link.
2. **Convert**: Transforms the JSON into `api_for_ai.md` (clean, no code samples, ready for ChatGPT/Claude/Gemini).

## Requirements
- Python 3.7+
- Internet connection

## Quick Setup
1. Clone or download the files.
2. Install dependencies (run once):
   ```bash
   pip install -r requirements.txt
